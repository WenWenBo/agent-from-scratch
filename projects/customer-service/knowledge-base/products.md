# 产品目录

## TinyBot Pro（智能助手旗舰版）

- **价格**: ¥299/月，¥2999/年（年付8.3折）
- **功能**: 无限对话、优先响应、自定义知识库、API 接入、团队协作
- **适用**: 企业用户、开发团队
- **限制**: 单次对话最大 50 轮，知识库最大 100MB

## TinyBot Lite（智能助手基础版）

- **价格**: ¥99/月，¥999/年（年付8.4折）
- **功能**: 每日 100 次对话、标准响应、基础知识库
- **适用**: 个人用户、小型团队
- **限制**: 每日 100 次对话，知识库最大 10MB

## TinyBot Free（免费体验版）

- **价格**: 免费
- **功能**: 每日 10 次对话、标准响应
- **适用**: 个人体验
- **限制**: 每日 10 次对话，不支持知识库，不支持 API

## TinyBot API

- **价格**: ¥0.01/次调用（前 1000 次免费）
- **功能**: REST API 接入、流式输出、工具调用
- **文档**: https://docs.tinybot.example.com/api
- **限速**: 60 次/分钟（可申请提升）
